Shopfies is a webpage design using php, CSS and Bootstrap that displays data through SQL statements. This is the final group project (4 people) for a class that I took in Fall 2021: TCSS 445 - Database System Design.

Through this project, I learned how to create database schema to store data. I also learned how about ER diagram and BCNF. We run DBMS on GCloud platform and LAMP Virtual Machine.
