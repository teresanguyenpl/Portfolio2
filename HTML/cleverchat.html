Cleverchat is an Android application that allows users to register, log in, add friends, chat with others and get real time weather updated. This is the final group project (4 people) for a class that I took in Fall 2021: TCSS 450 - Mobile Application Programming.

Through this project, I learned and implemented the login/register/contact systems using Node.JS (a back-end JavaScript runtime) and Heroku (a cloud application platform). More specifically, I and my team designed our databases with PostgreSQL, connected to web service HTTP, built different routes for handling the CRUD operation and tested our APIs using Postman. We also used external APIs like Pushy and OpenWeatherMap for our messaging and weather systems. For the front-end, we used Android Studio, with multiple recycler views and theme mode switching. This project not only gave me the opportunity to learn the top languages and frameworks for software development, but it also gave me more experience working with others.
